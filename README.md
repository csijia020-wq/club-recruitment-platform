# 社团招新智能匹配平台

这是一个基于 `Vite + React + TypeScript` 的高校社团招新智能匹配平台演示项目，包含：

- 评审首页：高校品牌、`PLAN.md` 入口、产品概览、三端入口
- 学生端：智能迎新画像、可解释推荐、社团主页、一键报名、进度追踪、通知中心
- 社团管理端：招新配置、申请看板、批量通知、数据看板
- 学校管理端：社团审核、全局数据、治理日历
- `PLAN.md` 文档页：应用内直接渲染仓库根目录文档

## 启动方式

```bash
npm install
npm run dev
```

默认访问地址通常为 `http://127.0.0.1:5173/`。

## 构建检查

```bash
npm run build
```

## 数据说明

- 演示数据位于 `src/data/mockData.ts`
- 本地状态会写入浏览器 `localStorage`
- 存储键升级为 `club-recruitment-demo-v2`
- 点击页面中的“重置演示数据”可以恢复默认演示态

## 说明

- 学校 logo 当前使用可替换的占位校徽
- `PLAN.md` 为单一文档来源，首页入口会跳转到应用内文档页
